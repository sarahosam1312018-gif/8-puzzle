# 8-puzzle
This project implements a solver for the 8-Puzzle problem using four search algorithms:

BFS (Breadth-First Search)

IDS (Iterative Deepening Search)

UCS (Uniform Cost Search)

A* Search with Manhattan Distance heuristic

# Project Structure
```bash
8-puzzle/
├─ src/
│  ├─ bfs_8puzzle.py
│  ├─ IDS.py
│  ├─ ucs_8puzzle.py
│  └─ 8-puzzle_using_A_star.py
```
# Requirements

 Python 3.7 or higher

# How to Run

Open a terminal in the project root folder and run any algorithm file directly:

```bash
python src/bfs_8puzzle.py
python src/IDS.py
python src/ucs_8puzzle.py
python src/8-puzzle_using_A_star.py
